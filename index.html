about.

My interest in coding started a few years ago after a friend of mine changed careers and worked as a Junior Back End Developer. I was curious to know what it was all about, and after trying out some online resources I became really attracted to the problem solving aspect of coding, and pretty quickly I was playing around with code a lot in my spare time. I started coding more and more in my free time, and exploring different resources.

One thing that I really loved when I started discovering more about the industry is its open source nature. I was surprised about how many resources and tools were available for free, and how keen people are to give you advice and share their knowledge. As I’m right at the start of my career in development it’s really important to me to continue to learn to work in an environment that supports my development.

Currently I'm working on a website for a friend's business, Proven Goods Co., as well as daily challenges on Wes Bos' Javascript30.

Chapter

A MERN stack application that allows a user to create and/or discover online book groups and participate in discussion around their favourite books. The app uses image recognition through the Google Vision API which allows the user to post a picture of a book and receive websites to buy the book or read reviews.

Highlight was working on the app was getting the API up and running and the response formatted on the front end. I also really enjoyed using React to build a component-based app.

HTML5 | CSS3 (SASS) | Mongoose | MongoDB | React | Node.js | Express | RESTful API | Git | Heroku

What's In My Fridge?

A MEAN stack application built over one week, planned using wireframes, with team contributions using GitHub. The app allows a user to search for recipes based on the ingredients they have at home already, from data supplied through a third party API, Spoonacular. My role involved API functionality, including user input as search query, running server side tests on the API, and displaying data on the search page and the individual recipe pages. I was also involved in the design, and was responsible for managing the project.

Highlight was working with a team. Coding had previously been something I'd only done alone so I was unsure about how to assign responsibilities. It turned out to be a great experience that I learnt a lot from.

HTML5 | CSS3 (SASS) | Mongoose | MongoDB | Angular.js | Node.js | Express | RESTful API | Git | Heroku
